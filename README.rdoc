====Ragnar's Rock
A tool for collboratively building a multimedia record of a trip through photos, video and audio.

=====WIP
These are early days. Very little is in place at this point. 

=====Planned Features
* batch import photos from 
  * facebook
  * google+
  * instagram
* embed audio (soundcloud, other?)
* display panorama images
* initial map location should be configurable
